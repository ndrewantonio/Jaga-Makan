# Jaga Makan

Jaga Makan is an android based application that can detect food images and display its calories and nutritional values.

Foods that are available for detection:

- Ayam Geprek
- Bakso Sapi
- Bubur Ayam
- Ketoprak
- Martabak Manis
- Mie Ayam
- Nasi Goreng
- Pempek
- Rendang
- Soto Ayam

Application developed using Flutter
